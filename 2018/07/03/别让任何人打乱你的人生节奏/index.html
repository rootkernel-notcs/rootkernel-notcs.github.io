别让任何人打乱你的人生节奏

之前有个视频火了，里面有个学生上去演讲，反对校长所作的演讲。

内容如下：

  • 有的人21岁毕业，到27岁才找到工作；有的人25岁才毕业，但马上就找到工作了；有的人没上过大学，却在18岁就找到热爱的事业；有的人一毕业就找到好工作，赚很多钱，却过得不开心；
  • 有的人在16岁就清楚地知道自己要什么，却在26岁时改变想法；有的人有了孩子，却还是单身；有的人结婚了，却过了十年才要孩子；有的人身处一段感情，爱的却是别人；有的人明明彼此相爱，却没有在一起。
  • 人生的每一件事都取决于我们自己的时间，你身边有些朋友也许遥遥领先于你，有些人也许落后于你，但凡事都有他自己的节奏，他们有他们的，你有的，耐心一点。
    ```

我们听过很多鸡汤，我们麻木了，我们还是逃不出生活的魔掌，我们曾经认为开心地活着就够了，但是我们不得不为各种事情出卖自己的时间，感情，金钱。

在人的一生中，我们每个时期都有不同的目标和理想，也会不断地变换目标，可能变高了，也可能变低了。这样我们的人生就逐渐地产生差距，有些人会在某些特定的时刻，特定的事情醒悟过来，又走上更好的道路，这或许就是人生，就是它上面所说的事情吧。

其实我觉得只要人生的主旋律没变，保持初心，砥砺前行，成为自己心中那个更好的自己，那就好了。
